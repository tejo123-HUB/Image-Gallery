# Stunning Image Gallery

A premium, highly responsive image gallery built with HTML, CSS, and Vanilla JavaScript. This project features a modern dark-mode aesthetic with glassmorphism effects, a built-in category filtering system, and an interactive lightbox for a seamless user experience.

## ✨ Features

- **Responsive CSS Grid Layout:** Adapts flawlessly to any screen size, from desktop monitors down to mobile devices.
- **Dynamic Category Filtering:** Instantly filter images by categories (Nature, Architecture, Technology) with smooth fade-in animations.
- **Interactive Lightbox:** Click on any image to view it in full screen.
- **Touch-Swipe & Keyboard Navigation:** Navigate through the lightbox using on-screen buttons, keyboard arrow keys, or mobile swipe gestures.
- **Premium Aesthetics:** Features an immersive animated radial gradient background, modern hover animations, and sleek glassmorphism (`backdrop-filter`) UI elements.

## 🛠️ Technologies Used

- **HTML5:** Semantic layout and structure.
- **CSS3:** Custom properties (variables), CSS Grid, Flexbox, animations, media queries, and premium styling.
- **Vanilla JavaScript (ES6):** DOM manipulation, event listeners for filtering, lightbox state management, and touch gesture handling.
- **FontAwesome:** For intuitive navigation icons.
- **Google Fonts (Inter):** Clean, modern typography.

## 🚀 How to Run Locally

Since this project uses plain HTML, CSS, and JavaScript, no complex build tools or local servers are required!

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/yourusername/image-gallery.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd image-gallery
   ```
3. **Open the project:**
   Simply double-click the `index.html` file in your File Explorer to open it in your default web browser, or serve it using a local development server (like VS Code Live Server) for automatic reloads.
